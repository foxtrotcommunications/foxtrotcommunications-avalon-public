# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Avalon, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please email: **customer_support@foxtrotcommunications.net**

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within **48 hours** and provide a detailed response within **5 business days**.

## Scope

This policy applies to the Avalon open-source repository. For vulnerabilities related to the Forge platform or hosted services, please contact customer_support@foxtrotcommunications.net separately.

## Supported Versions

| Version | Supported |
|---------|-----------|
| main    | ✅        |
| < main  | ❌        |

## Disclosure Policy

- We follow coordinated disclosure practices
- We will credit reporters (unless anonymity is requested)
- We aim to patch critical vulnerabilities within 72 hours of confirmation
